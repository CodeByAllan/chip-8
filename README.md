
# CHIP-8

A Chip-8 emulator written in Go, capable of running classic games from the 1970s with graphics, sound and user input support


## Documentation

[Wiki](https://en.wikipedia.org/wiki/CHIP-8)

[Instructions](https://github.com/mattmikolay/chip-8/wiki/CHIP%E2%80%908-Instruction-Set)

[Extra](https://multigesture.net/articles/how-to-write-an-emulator-chip-8-interpreter/)


## Stack

**Front-end:** [Raylib](https://www.raylib.com/)

**Back-end:** [Go](https://go.dev/)
